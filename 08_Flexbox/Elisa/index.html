Heute lernen wir endlich Flexbox

Was ist CSS Flexbox?

  • Flexbox ist eine moderne und einfach Möglichkeit responsive und flexible Layouts zu erstellen => wir brauchen keine feste Groessenangaben
  • Das Model artbeitet auf zwei Achsen auf denen die Inhalte verteilt werden
  • Flexbox ist für jede Bildschirmgroesse geeignet
  • Flexbox arbeitet mit einem Containerelement (flex Container) und den darin enthaltenen Kindelementen der 1. Ebene. flexbox wirkt genau eine Ebene tief -> weitere Verschachtelungen werden ignoriert
